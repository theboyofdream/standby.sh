# standby.sh

A clean, modern, and customizable standby clock and timezone manager built for your browser. Inspired by the iOS Standby mode.

<!--Live URL: standby-sh.vercel.app-->

## Features

- Large Digital Clock: A beautiful, easy-to-read clock face that's perfect for your desk or nightstand.
- Multiple Timezones: Add and manage up to 4 different timezones.
<!--- Countdown Timer: A simple and effective countdown timer.-->

### Customization:
- Toggle seconds visibility.
- Switch between 12-hour and 24-hour formats.
- Light & Dark mode support (with system preference detection).
- Modern UI: Built with shadcn/ui and Tailwind CSS for a sleek and responsive experience.
